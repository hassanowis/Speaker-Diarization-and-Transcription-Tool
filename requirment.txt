
# Instructions for Hugging Face Token
# 1. Create an account on Hugging Face: https://huggingface.co/
# 2. Navigate to 'Settings' -> 'Access Tokens'.
# 3. Create a new token with 'read' and 'write' access.
# 4. Accept the terms of license for the PyAnnote model: https://huggingface.co/pyannote/speaker-diarization-3.1
# 5. Replace 'your_huggingface_token' in your code with the generated token.

# Google Colab Setup Instructions
# 1. Upload the MP4 file to Google Colab or Google Drive.
# 2. Provide the path to the MP4 file in the script.
# 3. Run the script to execute the pipeline.
